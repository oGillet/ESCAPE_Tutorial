ESCAPE - Application web
  • Tutorial

Nous proposons ici un tutoriel pour permettre aux modélisateurs d’utiliser les outils ESCAPE via la plateforme de simulation GAMA et l’application web “ESCAPE_input_generator”. Ce tutoriel est composé de plusieurs étapes (générer les inputs du système, créer l’environnement du système, utiliser des outils ESCAPE, réaliser quelques statistiques sur le modèle d’évacuation).

Ce tutoriel présente le développement d’un système simulant de l’évacuation, fictive et peu probable, de la population de la commune du Genétay lors d’un feu de forêt.

Génerer les inputs du modèle

  • Se rendre sur l’application web ESCAPE
  • Télécharger les données d’intérêt pour votre système (shelter, itinéraire d’évacuation, point de fuite des agents)
    ⚠️ ⚠️ ⚠️ Quelques soient les données à télécharger, vous devez créer le shapefile de votre zone d’étude car il est nécessaire pour obtenir l’extension spatiale de la zone d’étude afin de télécharger les données OpenStreetMap. Vous pouvez retrouver ces coordonnées (Xmin, Xmax, Ymin, Xmin, EPSG::4326) en bas de la page web après création du polygone de la zone d’étude.

  • Vous pouvez par la suite télécharger vos données. Vous trouverez une archive composée de plusieurs shapefiles .shp et d’un fichier text .csv d’organisation des transports en commun.

Créer l’environnement du système

  • Ensuite, vous devez vous rendre sur le site web d’OpenStreetMap OpenStreetMap
  • Renseigner les coordonnées obtenues précédemment (Xmin, Xmax, Ymin, Xmin, EPSG::4326) et télécharger les données OpenStreetMap (routes et bâtiments).Vous obtenez un fichier au format .osm à utiliser via les outils ESCAPE pour générer l’environnement.
  • Les données téléchargées, OpenStreetMap comme ESCAPE, sont à extraire dans le répertoire include 📁 de votre projet ESCAPE.

Au total, vous devez créer trois fichiers .gaml pour exécuter votre modèle dont deux pour créer l’environnement d’éxecution du système.
1 - Un fichier pour créer votre environnement à partir du xml d’OpenStreetMap
2 - Un fichier pour calculer la matrice des plus courts chemins
3 - Un fichier contenant le script principal du modèle d’évacuation

Importer les résultats dans l’application

Vous devez uploader dans l’onglet output le fichier oputs_SHP_ESCAPE.txt.
